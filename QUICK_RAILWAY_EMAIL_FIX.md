# ⚡ Быстрое исправление: Email на Railway

## 🎯 Проблема
Локально коды приходят ✅  
На Railway коды НЕ приходят ❌

## 💡 Причина
Не настроены переменные окружения на Railway

## 🔧 Исправление (3 минуты)

### 1️⃣ Откройте Railway
https://railway.app → Ваш проект → Backend сервис

### 2️⃣ Перейдите в Variables
Вкладка **Variables** → **+ Add Variable**

### 3️⃣ Добавьте две переменные

```
EMAIL_USER = ваша-почта@yandex.ru
EMAIL_PASSWORD = пароль-приложения-yandex
```

⚠️ **ВАЖНО:** Используйте **пароль приложения**, НЕ обычный пароль!

### 4️⃣ Получите пароль приложения

#### Для Yandex:
1. Откройте: https://id.yandex.ru/security/app-passwords
2. Создайте пароль для "LADA Backend"
3. Скопируйте пароль

#### Для Gmail:
1. Откройте: https://myaccount.google.com/apppasswords
2. Включите 2FA (если выключена)
3. Создайте App Password
4. Скопируйте 16-значный пароль

### 5️⃣ Сохраните и подождите
Railway автоматически перезапустит сервис (1-2 минуты)

### 6️⃣ Проверьте логи
Railway → Deployments → View Logs

Должно быть:
```
✅ SMTP сервер готов к отправке писем
```

## ✅ Готово!
Теперь протестируйте запрос кода - должен прийти на почту!

- ✅ Прошло 1-2 минуты после сохранения
