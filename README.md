# Интернет-приемная АвтоВАЗ

MVP интернет-приемной ПАО «АвтоВАЗ» с возможностью публикации через GitHub Pages.

## Технологии

- **Frontend**: React 19
- **Framework**: Next.js 15
- **UI**: shadcn/ui + Tailwind CSS
- **TypeScript**: 5.6
- **Деплой**: GitHub Pages

## Функционал

- ✅ Авторизация через ЕСИА (демо-режим)
- ✅ Форма подачи обращения
- ✅ Категории обращений
- ✅ Прикрепление файлов
- ✅ Визуальный стиль АвтоВАЗ
- ✅ Адаптивный дизайн

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Экспорт статических файлов
npm run export
```

## Деплой на GitHub Pages

1. В настройках репозитория включите GitHub Pages:
   - Settings → Pages
   - Source: GitHub Actions

2. При пуше в ветку `main` автоматически запустится деплой

3. Сайт будет доступен по адресу: `https://[username].github.io/VAZ/`

## Структура проекта

```
.
├── app/                # Next.js App Router
```

## Особенности

- **ЕСИА**: Реализован демо-режим, принимает любые данные
- **Цветовая схема**: Фирменные цвета АвтоВАЗ (#0066B3, #004A8F)
- **GitHub Pages**: Настроен static export для публикации
