# Быстрый старт для мобильного тестирования

## ⚡ Вариант 1: С Production Backend (Рекомендуется)

Frontend автоматически подключается к production backend на Railway.

### Шаг 1: Запустите Frontend
```bash
npm run dev
```

### Шаг 2: Узнайте IP адрес вашего компьютера
```bash
ipconfig getifaddr en0
```
Пример результата: `192.168.1.100`

### Шаг 3: Откройте на телефоне
В браузере телефона откройте:
```
http://192.168.1.100:3000
```

**Важно**: Телефон и компьютер должны быть в одной Wi-Fi сети!

---

## 🔧 Вариант 2: С Локальным Backend

Если нужно тестировать с локальным backend:

### Шаг 1: Создайте .env.local
```bash
echo "NEXT_PUBLIC_API_URL=http://192.168.1.100:3001" > .env.local
```
**Замените `192.168.1.100` на ваш реальный IP адрес!**

### Шаг 2: Запустите Backend
```bash
cd backend
npm run dev
```

### Шаг 3: Запустите Frontend
```bash
cd ..
npm run dev
```

### Шаг 4: Откройте на телефоне
```
http://192.168.1.100:3000
```

## Что исправлено?
1. ✅ Backend слушает на всех сетевых интерфейсах (0.0.0.0)
2. ✅ Frontend ждет завершения запроса перед переходом на следующий шаг
3. ✅ Таймер работает при переключении вкладок браузера
4. ✅ Код не нужно вводить дважды при регистрации
5. ✅ Коды не перезаписываются при повторной отправке

## Тестирование
1. Нажмите "Получить код"
2. Дождитесь перехода на страницу ввода кода (0.5-2 секунды)
3. Проверьте email - код должен прийти
4. Введите код и продолжите

Если код не приходит - проверьте логи backend в терминале.
